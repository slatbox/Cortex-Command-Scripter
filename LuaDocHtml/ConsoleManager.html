ConsoleManager

The singleton manager of the lua console.

Has no parent class.


Function:

PrintString

Prints a string into the console.

Arguments:
The string to print.

Return value:
None.


Function:

SaveInputLog

Writes all the input strings to a log in the order they were entered.

Arguments:
The filename of the file to write to.

Return value:
None.


Function:

SaveAllText

Writes the entire console buffer to a file.

Arguments:
The filename of the file to write to.

Return value:
None.


Function:

Clear

Clears all previous input.

Arguments:
None.

Return value:
None.


Property:

ForceVisibility

Report whether the console is forced to be always visible.


Property:

ScreenSize

Gets how much of the screen that the console is covering when opened. The ratio of the screen that is covered: 0 - 1.0.
