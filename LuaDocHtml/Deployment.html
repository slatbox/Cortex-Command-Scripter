Abstract Class:

Deployment

A special SceneObject that specifies a Loadout of whatever Tech is relevant to be placed in a specific location in a Scene.

Parent: SceneObject


Property:

ClassName

The class name of this Entity. A string with the friendly-formatted type name of this object.


Function:

GetLoadoutName

Gets the name of the Loadout that this Deployment spawns.

Arguments:
None.

Return value:
The name of the Loadout preset that this Deployment spawns.


Read-Only Property:

SpawnRadius

The radius around this deployment that gets checked if another actor/item of the same type and name already exists and will block re-spawning a new one by this The radius this Deployment will be checking within.


Read-Only Property:

ID

Return this deployment's unique ID This deployment's ID


Read-Only Property:

HFlipped

Returns whether this MOSprite is being drawn flipped horizontally (along the vertical axis), or not. Whether flipped or not.
