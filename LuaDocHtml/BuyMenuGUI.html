Generic Class:

BuyMenuGUI

A full menu system that represents a purchasing GUI for Cortex Command

Has no parent class.


Function:

SetMetaPlayer

Sets which MetaPlayer uses this menu, if any.

Arguments:
The index of the MetaPlayer that uses this menu.

Return value:
None.


Function:

SetNativeTechModule

Sets which DataModule ID should be treated as the native tech of the user of this menu.

Arguments:
The module ID to set as the native one. 0 means everything is native.

Return value:
None.


Function:

SetForeignCostMultiplier

Sets the multiplier of the cost of any foreign Tech items.

Arguments:
The scalar multiplier of the costs of foreign Tech items.

Return value:
None.


Function:

SetModuleExpanded

Sets whether a data module shown in the item menu should be expanded or not.

Arguments:
The module ID to set as expanded.
Whether should be expanded or not.

Return value:
None.


Function:

LoadAllLoadoutsFromFile

Loads or re-loads all the loadout presets from the appropriate files on disk. This will first clear out all current loadout presets!

Arguments:
None.

Return value:
Success or not.


Property:

EnforceMaxPassengersConstraint

Tells whether passenger count constraints are enforced by this buy menu. True if passenger constraints are enforced by this menu, false otherwise


Property:

EnforceMaxMassConstraint

Sets whether mass constraints are enforced by this buy menu. True if mass constraints are enforced by this menu, false otherwise
