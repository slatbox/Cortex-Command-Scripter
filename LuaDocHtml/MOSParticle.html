Concrete Class:

MOSParticle

A small animated sprite that plays its animation and changes the animation and playback speed when it collides with other things.

Parent: MOSprite


Property:

ClassName

The class name of this Entity. A string with the friendly-formatted type name of this object.


Property:

Framerate

The current framerate of the animation playback. The current framerate in frames per second.


Read-Only Property:

IsGold

Indicates whether this MO is made of Gold or not. Whether this MovableObject is of Gold or not.
