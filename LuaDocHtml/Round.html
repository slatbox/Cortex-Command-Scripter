Concrete Class:

Round

A round containing a number of projectile particles and a one shell.

Parent: Entity


Property:

ClassName

The class name of this Entity. A string with the friendly-formatted type name of this object.


Read-Only Property:

NextParticle

The next particle contained in this Round, ownership is NOT transferred! A pointer to the next particle, or 0 if this Round is empty.


Read-Only Property:

Shell

The shell casing preset of this Round. Ownership IS NOT transferred! A pointer to the shell casing preset, or 0 if this Round has no shell.


Read-Only Property:

FireVel

The velocity at which this round is to be fired. A float with the velocity in m/s.


Read-Only Property:

ShellVel

The velocity at which this round's shell is to be ejected. A float with the shell velocity in m/s.


Read-Only Property:

Separation

The separation of particles in this round. A float with the separation range in pixels.


Read-Only Property:

ParticleCount

Returns how many particles are contained within this Round, not counting the shell. The number of particles.


Read-Only Property:

AILifeTime

Returns the lifetime of the projectile used by the AI when executing the shooting scripts. The life time in MS used by the AI


Read-Only Property:

IsEmpty

Returns whether this Round is out of particles or not. Whether this Round is out of particles or not.
