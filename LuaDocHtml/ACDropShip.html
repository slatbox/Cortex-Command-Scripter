Concrete Class:

ACDropShip

A hovering craft, with two engines on each attached on each end which tilt independently of the body to achieve steering.

Parent: ACraft


Property:

ClassName

The class name of this Entity. A string with the friendly-formatted type name of this object.


Read-Only Property:

RightEngine

The right side engine. An AEmitter pointer.


Read-Only Property:

LeftEngine

The left side engine. An AEmitter pointer.


Read-Only Property:

RightThruster

The right side secondary thruster. An AEmitter pointer.


Read-Only Property:

LeftThruster

The left side secondary thruster. An AEmitter pointer.


Read-Only Property:

LeftHatch

The left side hatch. An Attachable pointer.


Read-Only Property:

RightHatch

The right side hatch. An Attachable pointer.


Property:

MaxEngineAngle

Get max engine rotation angle in degrees. Max engine angle in degrees.


Property:

LateralControlSpeed

The abstract rate of LateralControl change. Default is 6 Current lateral control speed value.


Function:

DetectObstacle

Checks for obstacles in the travel direction.

Arguments:
How far ahead of travel direction to check for obstacles.

Return value:
Which MOID was detected as obstacle. g_NoMOID means nothing was detected.


Function:

GetAltitude

Gets the altitide of this' pos (or appropriate low point) over the terrain, in pixels.

Arguments:
The max altitude you care to check for. 0 Means check the whole scene's height.
The accuracy within which measurement is acceptable. Higher number here means less calculation.

Return value:
The rough altitude over the terrain, in pixels.
