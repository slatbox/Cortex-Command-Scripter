Concrete Class:

HeldDevice

An articulated device that can be weilded by an Actor.

Parent: Attachable


Property:

ClassName

The class name of this Entity. A string with the friendly-formatted type name of this object.


Read-Only Property:

SupportPos

The absolute position of the support handhold that this HeldDevice offers. A vector describing the absolute world coordinates for the support position of this HeldDevice.


Read-Only Property:

MagazinePos

The absolute position of the magazine or other equivalent point of this. A vector describing the absolute world coordinates for the magazine attachment point of this


Read-Only Property:

MuzzlePos

The absolute position of the muzzle or other equivalent point of this. A vector describing the absolute world coordinates for the muzzle point of this


Property:

MuzzleOffset

The unrotated relative offset from the position to the muzzle or other equivalent point of this. A unrotated vector describing the relative for the muzzle point of this from this' position.


Property:

StanceOffset

The current position offset of this HeldDevice's joint relative from the parent Actor's position, if attached. A const reference to the current stance parent offset.


Property:

SharpLength

Gets how much farther an Actor which holds this device can see when aiming this HeldDevice sharply. The length in world pixel units.
